# My Dashboard

A personal Electron desktop dashboard.

## Requirements
- Node.js installed (https://nodejs.org) — LTS version recommended

## How to run (dev mode)

1. Open a terminal in this folder
2. Run:
   ```
   npm install
   npm start
   ```

## How to build a .exe (Windows)

1. Run:
   ```
   npm install
   npm run build
   ```
2. Your `.exe` installer will appear in the `dist/` folder.

## Notes
- Tasks are saved locally in localStorage (persist between sessions).
- No internet connection required.
- The green sidebar color is #005e40.
